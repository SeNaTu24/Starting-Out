## 🚀 Getting Started

### Prerequisites
- Web browser

### Installation
1. Clone the repository
```bash
git clone https://github.com/SeNaTu24/Starting-Out.git
```

2. Navigate to project directory
```bash
cd Starting-Out
```

3. Open in browser
```bash
# Simply open index.html in your browser
```
